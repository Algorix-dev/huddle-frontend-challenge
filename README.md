# Huddle Landing Page

A solution to the [Huddle landing page challenge](https://www.frontendmentor.io/challenges/huddle-landing-page-with-a-single-introductory-section-B_2Wvxgi0) on Frontend Mentor.

## Built With

- Semantic HTML5
- CSS (desktop-first)
- Flexbox
- Google Fonts (Poppins, Open Sans)
- Font Awesome icons
- Responsive design with media queries

## What I Learned

- How to swap background images between mobile and desktop using media queries
- Mobile vs desktop-first CSS approaches
- Using flexbox for both page layout and component alignment
- Correct way to import Google Fonts via the googleapis.com CSS delivery URL
